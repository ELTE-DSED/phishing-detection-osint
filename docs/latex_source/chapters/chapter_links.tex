\section{Deployed Applications}

\subsection{Frontend Web Application}

The PhishGuard web interface is deployed on Vercel's Edge Network:

\begin{itemize}
    \item \textbf{URL:} \url{https://project-4soy4.vercel.app}
    \item \textbf{Platform:} Vercel (Serverless + Edge Network)
    \item \textbf{Features:}
    \begin{itemize}
        \item Responsive design for desktop, tablet, and mobile
        \item Light and dark mode support
        \item Real-time phishing analysis
        \item Analysis history with local storage
        \item Shareable result links
    \end{itemize}
\end{itemize}

\subsection{Backend API}

The REST API and ML inference engine is deployed on Render:

\begin{itemize}
    \item \textbf{URL:} \url{https://phishguard-api-upl2.onrender.com}
    \item \textbf{Platform:} Render (Docker containers)
    \item \textbf{API Documentation:} \url{https://phishguard-api-upl2.onrender.com/docs}
    \item \textbf{Features:}
    \begin{itemize}
        \item OpenAPI/Swagger documentation
        \item Real-time OSINT queries
        \item XGBoost ML inference
        \item NLP text analysis
        \item Health check endpoint at \texttt{/api/health}
    \end{itemize}
\end{itemize}
